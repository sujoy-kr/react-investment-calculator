# Investment Calculator

A React investment calculator for the Bangladesh. It calculates compound interest, yearly investment growth, and total returns, all localized with Bangla numerals.

[Live Demo Link](https://react-investment-calculator-k0hwjv8ao-sujoykrs-projects.vercel.app)

![App Screenshot](docs/app-screenshot.png)

## Setup

```bash
npm install
npm run dev
```
